# Ice Cream Parlour Management System
A project developed for managing ice cfeam parlour operations.

## Features
- Developed a web-based system to manage billing, order processing, managing daily sales record, and 
  product management. 
- Designed user-friendly and responsive interfaces with efficient backend data handling. 
- Improved operational efficiency and reduced manual errors through automation of customer orders 
  and inventory management.

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL
